集成 PanleSDK

  1. 创建工程

    在 Android Studio 中建立你的工程,接入公版 SDK 并配置完成

  2. 根目录的 build.gradle 配置

    allprojects {
   repositories {
       maven { url 'https://maven-other.tuya.com/repository/maven-releases/' }
       maven { url 'https://jitpack.io' }
   }
}
